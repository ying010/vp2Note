  • 程序计数器作用：记住下一条JVM指令的执行地址

物理上通过寄存器实现

  • 特点：
    • 程序计数器是线程私有的
    • 程序计数器不会出现内存溢出

# 1.2 栈

  • **定义 **：线程运行需要的内存空间，由栈帧组成

    • 栈帧：每个方法运行时需要的内存
    • 每个线程只能由一个活动栈帧，对应当前正在执行的方法

  • 垃圾回收只回收堆内存，不涉及栈内存

# 1.2.1栈内存溢出

异常： java.lang.StackOverflowError

  • 栈帧过多导致
    • 循环调用
      • 没有正确关闭的递归
      • 三方库导致
  • 栈帧过大
